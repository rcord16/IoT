Management Radio on Startup

When enabled, internal management radio setups an open wireless network called {model}:{mac}. This network will be available for 15 minutes after startup. It is very useful for initial device installation using a mobile device (UISP Mobile app is recommended). Keep in mind that even though this network is designed to cover very short distances, it can create an additional attack vector for determined attackers.

Note: airView feature cannot work while management radio is active because both features use the same radio.
